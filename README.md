# 如意管理系统

现代东方台账风格的 React 管理后台，以暖灰纸色、墨色排版和朱砂强调色呈现关键指标与系统动态。

前端项目位于 [`frontend/`](./frontend)，完整的页面说明、运行命令、目录结构和截图约定请参阅 [`frontend/README.md`](./frontend/README.md)。

## 快速开始

```bash
cd frontend
npm install
npm run dev
```

默认访问地址：`http://localhost:5173/dashboard`

## 质量检查

```bash
cd frontend
npm run lint
npm run build
```
